## 📌 About the Project
Cosmic Scrapbook is a digital portfolio designed to represent my personal journey, skills, and projects in a creative and interactive way. It serves as a central place where I showcase my work, interests, and technical abilities.

The design focuses on simplicity, smooth UI, and a modern aesthetic inspired by creative web experiences.

- Framer (No-code website builder)
- UI/UX design principles
- Responsive web design

## 🎯 Features
- Clean and modern UI
- Responsive design for all devices
- Smooth navigation and transitions
- Project showcase section
- About & skills presentation

This portfolio was created to:
